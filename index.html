Quick Installation & Setup

Setup the primary C++ shell client environment on your native Lubuntu workstation.

1. Install Dependencies & Build System:

# Install build requirements, Qt5, and WebEngine dependencies
sudo apt update
sudo apt install -y build-essential qt5-qmake qtbase5-dev qtdeclarative5-dev qtwebengine5-dev

2. Compile the Core Binary:

# Generate Makefile and compile Anodyne OS shell executable
qmake AnodyneOS.pro
make -j$(nproc)

3. Trigger TPM & zRAM System Formats:

# Execute cryptographic verity configurations
sudo ./scripts/setup-zram.sh
sudo ./scripts/setup-dm-verity.sh
